Descompacte a pasta.

Abra o VS Code.

O VS Code provavelmente vai perguntar se confia nos autores, diga que Sim.

Abra o arquivo app.py.

(Apenas na primeira vez) Abra o terminal do VS Code (Ctrl + J) e digite:

pip install -r requirements.txt
(Isso instala tudo o que o código precisa para rodar perfeitamente).

Pronto! Agora é só clicar no botão de Play (▷) no canto superior direito.

Código do prototipo UI/UX feito no figma: https://www.figma.com/design/I6PXeEkZrJSGNxZZZoMzNf/Projeto-Semestral?node-id=0-1&p=f&t=S1DLMEzeo0urixYZ-0
